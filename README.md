# 3D Platformer

A 3D platformer game built in Godot Engine using GDScript, where a robot 
must collect all coins while navigating obstacles and dealing with enemies.

## Gameplay
- Collect all coins to complete the level
- Navigate through obstacles and platforms
- Avoid or squish enemies to progress

## Tech Stack
- Godot Engine
- GDScript
- Blender (3D assets)

## How to Play
1. Download and install [Godot Engine](https://godotengine.org/)
2. Clone this repository
3. Open the project in Godot and hit Play

## Academic Context
Developed as a personal project to explore game mechanics and 
object-oriented programming in a 3D environment.
